## Security Flow
### 1. Account Creation: 
Master password hashed and stored safely.
### 2. Login: 
Master password checked via bcrypt for authentication.
### 3. Password Storage: 
All passwords are encrypted before storing.
### 4. Data Access: 
Passwords decrypted only using a securely hashed key.
